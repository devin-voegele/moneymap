## üõ†Ô∏è Technical Specifications

### For All Videos:

**Format:**
- Aspect Ratio: 9:16 (vertical)
- Resolution: 1080x1920 pixels
- Frame Rate: 30 fps
- File Format: MP4 (H.264 codec)
- Max File Size: 50MB for TikTok, 100MB for Instagram

**Text:**
- Always readable on mobile (minimum 48pt font)
- High contrast (white on dark, dark on light)
- Avoid small text
- Use bold fonts for impact

**Branding:**
- MoneyMap logo appears in last 3-5 seconds
- Website URL always visible in final frame
- Color scheme consistent with brand (Blue #3B82F6, Purple #8B5CF6)

**Captions:**
- Always include auto-generated captions
- Use for accessibility
- Helps with watch time (80% watch without sound)

**Music:**
- Royalty-free or licensed
- Volume: -12dB to -6dB (not too loud)
- Fade in/out smoothly
